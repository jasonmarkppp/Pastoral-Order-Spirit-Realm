🌾 田园智序 · 灵境小院 (Pastoral Order · Spirit Realm)
一个结合了山水田园画风与**“反拖延”AI 规则干预体系**的沉浸式全栈工作台。
在这里，你的每一次专注、每一次打卡，都会化作清泉与甘草，滋养你的心灵小院。

✨ 项目简介
“田园智序 · 灵境小院” 旨在打破传统效率工具带来的焦虑感。它将待办事项、习惯养成、番茄钟与电子宠物养成无缝融合。更重要的是，内置的 AI 智析引擎 会静默分析用户的数据库记录，在用户出现拖延迹象（如：连续中断打卡、任务大面积逾期）时，以温暖的方式给予提醒和目标降级建议，做你最贴心的效率管家。

🎯 核心特性
🌅 动态山水画 UI：根据真实时间自动切换清晨、午后、黄昏与深夜模式，星空与飞鸟动态渲染。

🎮 正向反馈游戏化：完成待办、坚持习惯、专注番茄钟可随机获得「水💧」「胡萝卜🥕」「草料🌿」，用于投喂小院里的向日葵、小羊和兔子。

🧠 AI 智析引擎 (规则驱动)：

断联干预：识别连续 3 天未打卡的习惯，主动询问并建议降低频率。

动态频率调整：7天完成率低于 30% 时，建议调整目标以符合当前节奏。

逾期监控：自动计算逾期任务，重点标红提醒优先处理。

📦 开箱即用 (桌面级体验)：提供一键打包与 .bat 启动脚本，实现无边框、沉浸式的本地桌面软件体验。

🛠️ 技术栈
前端：HTML5, CSS3, Vanilla JavaScript (原生开发，零依赖，极速加载)

后端：Python, Flask (轻量级 API 驱动)

数据库：SQLite3 (本地化存储，保护隐私)

打包工具：PyInstaller (转化为独立 EXE 运行)
